✍️ Grammaire : Décrire les animaux

Use these words to describe how animals look or act!

📏 Size & Power

Grand (Big) - L'éléphant est grand.
Petit (Small) - Le chat est petit.
Fort (Strong) - Le lion est fort.

🏃 Speed & Behavior

Rapide (Fast) - Le lapin est rapide.
Lent (Slow) - La tortue est lente.
Mignon (Cute) - Le chien est mignon.

❓ Répondre aux questions : Oui, Non... ou Si ?

How to answer "Yes" and "No" correctly in French!

Oui (Yes) "Tu aimes le chien ?" -> "Oui, j'aime le chien !"
Non (No) "Tu es un chat ?" -> "Non, je suis un enfant !"
Si (Contradictory Yes) "Tu n'aimes pas le lion ?" -> "Si, je l'adore !"

💡 Note: In French, we use "Si" instead of "Oui" when someone asks a negative question (using ne... pas) and we want to say "Yes, I actually do!"
